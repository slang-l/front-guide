# SEO 基础

SEO 的目标是帮助搜索引擎理解页面内容，并让用户在搜索结果中获得准确预期。前端主要负责结构、元信息、性能和可访问性。

## 基础要点

- 每个页面只有一个清晰的 `h1`。
- 使用准确的 `title` 和 `meta description`。
- 图片提供有意义的 `alt`。
- 链接文本表达目标内容。
- 保持页面加载速度和移动端体验。

## 示例

```html
<title>Front Guide - 前端学习指南</title>
<meta name="description" content="系统学习现代前端开发知识。" />
```

## 常见误区

不要堆砌关键词。好的 SEO 首先来自真实、清晰、有结构的内容。
