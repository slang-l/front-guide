# 语义化标签

语义化是指使用符合内容含义的 HTML 标签，而不是全部依赖 `div` 和 `span`。它能提升可读性、可访问性和搜索引擎理解页面的能力。

## 常见标签

- `header`：页面或区块头部。
- `nav`：导航链接区域。
- `main`：页面主体内容。
- `article`：独立内容单元。
- `section`：主题明确的内容分组。
- `aside`：补充信息。
- `footer`：页面或区块底部。

## 实践建议

先从内容结构出发，再选择标签。不要为了语义而嵌套过深，也不要把语义标签当成样式工具。

```html
<main>
  <article>
    <header>
      <h1>Flex 布局完全指南</h1>
      <p>系统学习 Flex 的核心概念。</p>
    </header>
  </article>
</main>
```

## 面试角度

语义化的价值通常可以从可维护性、可访问性、SEO 和团队协作四个角度回答。
