# 高频八股题

八股题的核心是概念表达。回答时先给结论，再补充原理、示例和实践经验。

## 示例问题

### 解释闭包

闭包是函数和其词法环境的组合。即使外层函数执行结束，内部函数仍然可以访问外层变量。

### Promise 和 async/await 的关系

`async/await` 是 Promise 的语法糖，让异步代码更接近同步写法，但底层仍然基于 Promise 和微任务。

### 什么是重排和重绘

重排会重新计算元素几何信息，重绘只更新视觉样式。重排成本通常更高。
