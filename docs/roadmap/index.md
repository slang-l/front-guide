# 前端学习路线

这条路线适合希望系统补齐前端能力的学习者。每个阶段都建议边学边写小项目。

## 第一阶段：页面基础

- HTML 语义化。
- CSS 盒模型、Flex、Grid。
- 响应式布局。

## 第二阶段：JavaScript 核心

- 数据类型和作用域。
- 闭包和原型。
- Promise 和 Event Loop。

## 第三阶段：TypeScript 与框架

- TypeScript 基础类型和泛型。
- Vue 或 React 组件开发。
- 路由、状态管理和接口封装。

## 第四阶段：工程化

- Vite 构建和开发服务器。
- ESLint、Prettier、Git 工作流。
- GitHub Actions 自动部署。

## 第五阶段：面试复习

- 高频手写题。
- 浏览器原理。
- 项目亮点表达。
- 简历和模拟面试。
